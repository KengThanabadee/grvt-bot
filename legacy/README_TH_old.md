# บอท GRVT Demo

ตัวอย่างของบอทเทรดที่ยืนหยัดสำหรับการแลกเปลี่ยน GRVT (Testnet/Prod) บอทนี้สาธิตวิธีการเชื่อมต่อ ตรวจสอบสิทธิ์ สร้างสัญญาณ และดำเนินการเทรดโดยใช้สำเนาท้องถิ่นของ GRVT Python SDK

## โครงสร้างโครงการ

- **`main.py`**: จุดเข้าของแอปพลิเคชัน ทำการรันลูปหลักซึ่งดึงข้อมูลสัญญาณและดำเนินการสั่งซื้อ
- **`config.py`**: ไฟล์การกำหนดค่าสำหรับข้อมูลรับรอง (API Key, Private Key, Account IDs) และการตั้งค่าการเทรด (Symbol, Leverage, Order Size)
- **`execution.py`**: จัดการการเชื่อมต่อกับ GRVT การตรวจสอบสิทธิ์ และการวางคำสั่ง ห่อหุ้มการโต้ตอบ `pysdk`
- **`trading_logic.py`**: มีตรรมชาติของกลยุทธ์ ปัจจุบันใช้เครื่องสร้างสัญญาณแบบสุ่มอย่างง่ายเพื่อวัตถุประสงค์ของการสาธิต
- **`requirements.txt`**: รายการพึ่งพา Python ที่จำเป็นในการเรียกใช้บอท
- **`pysdk/`**: สำเนาท้องถิ่นของ GRVT Python SDK เพื่อให้มั่นใจว่าบอทยืนหยัด

## การตั้งค่า

1.  **ข้อกำหนดเบื้องต้น**: ตรวจสอบให้แน่ใจว่าคุณมี Python 3.8+ ติดตั้งไว้

2.  **ติดตั้งพึ่งพา**:
    นำทางไปยังไดเร็กทอรีนี้และติดตั้งแพ็คเกจที่จำเป็น:
    ```bash
    pip install -r requirements.txt
    ```

3.  **การกำหนดค่า**:
    เปิด `config.py` และปฏิบัติตามหมายเหตุอย่างเข้มงวดเพื่อป้อนข้อมูลรับรอง GRVT ของคุณ
    ```python
    # ตัวอย่าง config.py
    GRVT_ENV = "testnet" # "testnet" หรือ "prod"
    GRVT_API_KEY = "your_real_api_key"
    GRVT_PRIVATE_KEY = "your_real_private_key"
    # ...
    ```

## การรันบอท

ในการเริ่มต้นบอท เพียงรัน `main.py` script:

```bash
python main.py
```

บอทจะ:
1. เริ่มต้นการเชื่อมต่อกับ GRVT
2. เข้าสู่ลูปตรวจสอบสัญญาณทุกวินาที
3. เมื่อสร้างสัญญาณ (สุ่มในการสาธิตนี้) มันจะวางคำสั่งตลาดบนสัญลักษณ์ที่กำหนดค่าไว้
4. บันทึกจะถูกพิมพ์ไปยังคอนโซลและบันทึกไว้ที่ `grvt_bot.log`

## การปรับแต่ง

- **เปลี่ยนกลยุทธ์การเทรด**: แก้ไข `trading_logic.py` เพื่อใช้อัลกอริทึมการเทรดของคุณเอง (เช่น MA crossover, RSI)
- **เปลี่ยนการดำเนินการ**: แก้ไข `execution.py` หากคุณต้องการประเภทคำสั่งขั้นสูง (Limit, Stop Loss เป็นต้น) หรือการจัดการมาร์จินที่ต่างกัน

## การทดสอบ

ฉันได้รวมโฟลเดอร์ `tests` ที่มีการทดสอบเชิงบูรณาการซึ่งใช้ข้อมูลรับรอง API จริงของคุณเพื่อตรวจสอบการทำงาน

**คำเตือน**: การทดสอบเหล่านี้จะดำเนินการสั่งซื้อ จริง ในสภาแวดล้อมที่ระบุในใน `config.py` ใช้สภาแวดล้อม `testnet` เพื่อหลีกเลี่ยงการสูญเสียเงินจริง

ในการรันการทดสอบ:
```bash
pytest grvt_demo_bot/tests
```
